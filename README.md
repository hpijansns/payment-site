# 💳 UPI Payment Website

This is a simple UPI-based payment website hosted on GitHub Pages.

## 🚀 Features
- Dynamic Amount Support
- QR Code Payment
- PhonePe Deep Link (Direct App Open)
- UPI Payment Support (GPay, Paytm, etc.)
